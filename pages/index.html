企业后台页面集

这里包含 5 类常见页面：角色首页、统计看板、列表管理、详情配置和审批长表单。 所有页面都复用了统一主题变量、边框、卡片和按钮风格，便于后续继续转成 Vue 页面或 Figma 设计稿。

白底后台 Element 风格兼容 桌面端优先

角色分发首页

适合系统首页、管理台首页、按角色进入不同工作台的入口型页面。

工单统计看板

双栏看板，涵盖指标卡、趋势图、排名列表、项目统计和关键词云。

列表管理页

搜索、工具栏、表格、行内操作与弹窗能力齐全的标准后台列表页。

详情配置页

双列设置 + 富文本编辑 + 附件上传 + 预览与规则提醒的文档型编辑页。

基础控件规范页

按 Element UI 规范补强后的按钮、输入、选择、标签、表格、弹窗和分页演示页。

审批长表单页

带右侧锚点导航、状态块、台账式表格和审批流区块的正式业务单据页面。
